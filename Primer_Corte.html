En el primer corte de Programación Funcional en Python vimos los siguientes conceptos:

Programación funcional: un paradigma de programación que se centra en el uso de funciones puras y composición.

Funciones puras: funciones que no tienen efectos secundarios, es decir, que no modifican el estado del programa.

Funciones de orden superior: funciones que pueden tomar otras funciones como argumentos o devolverlas como resultado.

Funciones lambda: funciones anónimas que se pueden usar en lugar de funciones definidas explícitamente.

Funciones reduce, map y filter

Las funciones reduce, map y filter son funciones de orden superior que nos permiten aplicar una función a los elementos de una secuencia de forma eficiente.

reduce(): aplica una función iterativamente a los elementos de una secuencia, produciendo un único valor como resultado.

map(): aplica una función a cada elemento de una secuencia, produciendo una nueva secuencia con los resultados.

filter(): devuelve una nueva secuencia con los elementos de una secuencia que cumplen una condición.
